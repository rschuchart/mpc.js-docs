getFilesInDirectory

  • getFilesInDirectory(directoryPath: string, recurseLevels?: number): Promise<MPDMusicFile[]>
    • Defined in mpc.ts:126

    Get the music files from the directory with the given path and its subdirectories.

    Parameters

    • directoryPath: string
    • Default value recurseLevels: number = 0

      The number of directory levels to recurse into. If no number (or 0) is given, only music files from the top directory are returned, if 1 is given, music files from the top directory and its immediate subdirectories are returned, and so on. If a negative number is given, the recursion is unlimited.

    Returns Promise<MPDMusicFile[]>
